# Digital IT
## Note
前提として、SFCCは他の複数のシステムと連携しており、不具合発生時にはその影響範囲を特定することが最初の関門となる。
関係するシステムは以下の通り。
- SFCC (Salesforce Commerce Cloud)
- HIP/ESB
- MDM
- Cegid
- Adobe Campaign

そして連携しているデータは以下の通り。
- 商品データ
- 在庫データ
- 受注データ
- 顧客データ
- プロモーションデータ
- 決済データ
- ポイントデータ


## Observation
現状3つの経路がある。
1. 他チームからの報告（異常発生）
2. 毎朝のアラートメール（SFCCからのメール）

これを受けて、以下の対応を行う。
1. トリアージ、影響範囲の特定
2. 関係者へのエスカレーション
3. 暫定対応
4. 原因調査と復旧対応
5. 再発防止策の検討と実施
